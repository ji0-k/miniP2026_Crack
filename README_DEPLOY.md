# CRACK 프로젝트 배포 가이드

이 패키지는 `Crack` 프로젝트의 전체 소스 코드, 설정(secrets), 그리고 업로드된 미디어 파일을 포함하고 있습니다.

## 🚀 빠른 시작 (원클릭 실행)

### Windows 환경
- `ONE_CLICK_START.bat` 파일을 더블 클릭하여 실행하세요.
- 자동으로 가상환경을 생성하고 필요한 라이브러리를 설치한 후 서버를 시작합니다.

### Linux (Ubuntu 등) 환경
1. 터미널을 엽니다.
2. 다음 명령어를 입력하여 실행 권한을 부여하고 스크립트를 실행합니다:
   ```bash
   chmod +x setup_and_run.sh
   ./setup_and_run.sh
   ```

## 📋 주요 정보
- **서버 주소**: `http://localhost:8012` (또는 서버 IP:8012)
- **포함 내용**:
  - `.git`: 전체 버전 관리 이력
  - `secrets/`: DB 접속 정보 등 비밀 설정
  - `uploads/`: 기존에 업로드된 모든 이미지 및 비디오
  - `static/`: 모델 파일(`best.pt` 등) 및 정적 자산

## ⚠️ 주의 사항
- 서버 컴퓨터에 **Python 3.8 이상**이 설치되어 있어야 합니다.
- 데이터베이스(MySQL 등)가 `secrets/` 설정에 맞게 준비되어 있어야 합니다.
- 포트 `8012`가 이미 사용 중인 경우, 스크립트가 기존 프로세스를 종료하려고 시도합니다.
